# Demo Recording Steps

1. Start the server and client.
2. Log in at `http://localhost:5173/login` using `agent / agent123`.
3. Create a new session.
4. Copy the invite link.
5. Join the session as the agent.
6. Open the invite link in another browser profile as the customer.
7. Allow camera and microphone access in both browsers.
8. Confirm both videos appear.
9. Send chat messages from both sides.
10. Mute audio and turn camera off/on once.
11. End the session from the agent room.
12. Return to the dashboard and show session history.
